RhizomerEye is a web application for interactive exploration of the semantic linked data available from SPARQL endpoints.

It automatically generates a user interface based on a tag cloud for all classes and a faceted view for each class. Each facet corresponds to a property and allows filtering class instances based on the property values.

For instance, you can explore the datasets:

  • Game of Thrones
  • DBPedia

If interested in testing Rhizomer with your own data, you can request and account: contact@rhizomik.net

It is also possible to deploy your own copy of Rhizomer. The source code and the instructions to do so are available from GitHub:

  • Client: Angular application consuming the Rhizomer API.
  • Server: Spring Boot application providing a REST API for datasets exploration.
